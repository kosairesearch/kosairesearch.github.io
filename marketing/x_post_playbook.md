# KOSAI — X 원글(Original Post) 플레이북
> 목표: 30일 내 100만 노출(reach). 답글 톤과 별개. 영어/글로벌 FinX 대상.

## 0. 답글 vs 원글 (톤이 다르다)
- **답글**: 짧고 대화체, 큰 계정에 "섞여들기", 한 가지 각, 링크 없음. (사람처럼, 살짝 캐주얼)
- **원글(이 문서)**: 브랜드 권위 + 도달 유발. 깔끔·자신감·데이터 주도. 캐주얼/슬랭 X, 코퍼릿 fluff도 X.

## 1. 원글 보이스
- "한국 주식을 영어로 다루는 유일한 소스"라는 권위.
- 형용사가 아니라 **사실/숫자**로 주목을 산다.
- 깔끔한 완성 문장이되 짧고 강하게. 과장어·이모지 도배·AI 정형구 금지.

## 2. 도달 잘 나오는 5가지 포맷
1. **Stat Banger** (단독 + 차트): 충격적인 숫자 하나. 예) "Two stocks are now half of Korea's entire market." → 리포스트·북마크 폭발.
2. **Map / List** (저장각): "Korea's AI hardware chain — the names you can't find in English:" → 북마크.
3. **Thread** (훅 + 짧은 5~7포인트 + 마무리): 한 가지를 깊게. 체류시간↑ → 부스트.
4. **Contrarian Take**: "Everyone says X. The data says Y."
5. **Name-to-know Story**: 글로벌이 아는 한국 것(불닭/삼양, K뷰티)을 금융으로. 비금융 크로스오버 도달.

## 3. 훅 공식 (1번째 줄 = 전부, 모바일은 1줄만 보임)
- 충격 숫자: "Two stocks = 52% of Korea's market."
- 호기심 갭: "You've eaten this company's product. You can't name the stock."
- 역발상: "The 'Korea discount' isn't a meme. Here's the receipt."
- 이해관계: "If memory margins roll over, half the KOSPI goes with it."
- 비공개성: "The chart nobody in the West is showing on Korea:"

## 4. 구조 규칙
- 1번째 줄 = 훅, 단독으로 작동해야 함.
- 짧은 문단/줄바꿈 (2~5줄), **한 글 = 한 아이디어**.
- 형용사보다 숫자.
- 데이터 글엔 **항상 차트/이미지** 첨부 (도달·체류↑).
- **본문에 외부 링크 금지** → 링크는 셀프리플에 (링크는 도달 깎음).
- 스레드는 끝에 가벼운 질문 or 페이로프로 마무리 (답글 유발 = 가중치 큼).
- 길이: Stat banger 1~3줄 / Thread 5~8개.

## 5. 피해야 할 것 (신뢰·도달 깎임)
- AI 티: "it's worth noting / in conclusion / let's dive in", em-dash 남발, 완벽 대칭 문장, 숫자마다 "~".
- 인게이지먼트 베이트: "RT if...", "follow for more" (페널티).
- 과장어: "game-changer, explode, to the moon".
- 매수/매도 단정 (컴플라이언스 — 자문업 아님). 사실 진술 + 질문으로.
- 이모지 도배 (0~2개).

## 6. 30일 케이던스 (100만 도달용)
- **원글 1개/일** (Stat / Thread / Name-story 번갈아).
- **가치 답글 5~10개/일** ← 0팔로워의 진짜 도달 엔진(큰 계정 도달 빌리기).
- **주 2~3회 스레드 스윙** (스레드가 바이럴 천장 제일 높음).
- 발행: **미국 활동시간(약 22~24시 KST) 예약.**
- 데이터 글엔 차트 필수.
- 링크 UTM → GA로 되는 포맷에 2배 투자.

## 7. 정직한 도달 산수
- 0팔로워 원글: 초기엔 글당 수백~수천 노출.
- 100만/30일은 주로 ①답글 폭격(빌린 도달) ②터지는 스레드 1~2개 ③일관성 복리 에서 나옴.
- 즉 **원글 = 신뢰 구축 + 프로필 방문자 전환 / 답글·스레드 히트 = 도달 숫자.** 둘 다 필요.

## 8. KOSAI 전용 콘텐츠 광맥 (데이터 해자)
- 시장 집중도(삼성+하이닉스 = 코스피 절반)
- 코리아 디스카운트(저PER 대형주)
- AI/반도체 공급망(하이닉스 외 숨은 이름)
- 글로벌 인지 크로스오버: 불닭(삼양), K뷰티(APR/실리콘투/코스맥스)
- 지주사 NAV 디스카운트
- 아무도 영어로 안 다루는 소형주 outlier
